Shadowsocks的安装与配置

时间： 2021-01-22   |   分类： 技术     |   阅读： 248 字 ~1分钟

环境准备

  • OS: CentOS 7.6

安装Shadowsocks

连接服务器,安装Shadowsocks

# 安装pip
yum install python-setuptools
easy_install pip
# 安装shadowsocks
pip install shadowsocks

配置服务文件

进入/opt目录，创建shadowsocks目录

cd /opt && mkdir shadowsocks

创建配置文件 shadowsocks.json

{
    "server":"0.0.0.0",
    "local_address": "127.0.0.1",
    "local_port":1080,
    "port_password":{
	"18001":"your-password"      //ssr连接端口(18001)与密码-注释内容删除
	},
    "timeout":300,
    "method":"aes-256-cfb",
    "fast_open": false
}

使用nohup启动ssr服务

nohup ssserver -c /opt/shadowsocks/shadowsocks.json -d start &

停止 ssr服务

ssserver -c /opt/shadowsocks/shadowsocks.json -d stop

使用Shadowsocks客户端连接

在Github等搜索下载shadowsocks

进行如下配置，完成即可！
